В данной программе реализовано приложение для тестирований на Python + QtDesigner с 3 уровнями пользователей: учитель, ученик, админ. Для каждого пользователя свой функционал.
Администратор можт регистрировать новых пользователей, учитель может создать новый тест, добавить вопрос в уже существующий тест и смотреть результаты тестирований учеников, а ученики проходить тестирование. 
Количество попыток тестирования ограничено, итоговый результат по лучшей попытки. В тесте могут быть вопросы с 2-6 вариантами ответов и с 1 или несколикими правильными.
